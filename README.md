# web-project

## Description
This webpage is about my journey with anime, from the first show I watched (Dragon Ball Z) to my current favorite (Jujutsu Kaisen). It includes a table of the anime I watched, a list of other favorites, images, and links to explore more. In Project 3, I built on this by adding CSS to improve the layout, styling, and overall look of the page.

## What I Learned
I learned how to structure a webpage with HTML, including using headings, paragraphs, images, lists, tables, and links. I also practiced adding a navigation bar and using HTML attributes like `bgcolor` and `<font>` for colors.

In Project 3, I learned how to use external CSS to style my webpage instead of relying on HTML. This included changing background and text colors, styling headers and paragraphs, centering elements, and adding hover effects to links, images, and list items. I also learned how CSS overrides inline styles and how to organize my layout more cleanly.

## Challenges
Some challenges included:
- Making sure the text was visible on a black background.
- Learning the proper placement for the navigation bar.
- Understanding how to add color and background using HTML-only techniques.

Additional challenges in Project 3:
- Figuring out how to link my external CSS file correctly.
- Understanding how CSS overrides HTML styling.
- Aligning images side-by-side and keeping them the same size.
- Styling elements like tables and lists without making the page look cluttered.
