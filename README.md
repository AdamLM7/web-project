# web-project

## Description
This webpage is about my journey with anime, from the first show I watched (Dragon Ball Z) to my current favorite (Jujutsu Kaisen). It includes a table of the anime I watched, a list of other favorites, images, and links to explore more. In Project 3, I built on this by adding CSS to improve the layout, styling, and overall look of the page.

## What I Learned
I learned how to structure a webpage with HTML, including using headings, paragraphs, images, lists, tables, and links. I also practiced adding a navigation bar and using HTML attributes like bgcolor and font for colors.

In Project 3, I learned how to use external CSS to style my webpage instead of relying on HTML. This included changing background and text colors, styling headers and paragraphs, centering elements, and adding hover effects to links, images, and list items. I also learned how CSS overrides inline styles and how to organize my layout more cleanly.

## Project 4 (JavaScript - Scouter Power Level System)

In Project 4, I took the original anime-themed webpage and turned it into an interactive “Scouter Power Level System” inspired by Dragon Ball Z. Instead of just displaying information, the page now actually works like a simple tool where users can input stats and get a calculated power level with a classification.

The scanner section uses a form where users enter values like strength, speed, endurance, energy, and stamina. JavaScript takes those inputs, adds them together, and then uses conditional statements to assign a power level tier based on the total.

I also improved the layout using CSS Grid and Flexbox to keep everything organized and centered. The page is fully responsive so it works on both mobile and desktop screens. I hosted the project using GitHub Pages, which helped me understand how file paths and folder structure affect whether CSS and JavaScript files load correctly.

## What I Learned (Project 4)

In this project, I learned how to make a webpage interactive using JavaScript. I learned how to:
- Get user input from form fields
- Use event listeners to run code when a button is clicked
- Do calculations with JavaScript
- Update the page dynamically based on user input
- Use if/else statements to create a tier system for power levels

I also got more practice using CSS Grid and Flexbox together to structure a layout. On top of that, I learned how important correct file paths are when linking CSS and JavaScript, especially when uploading a project to GitHub Pages.
