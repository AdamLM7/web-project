# web-project

## Description
This webpage is about my journey with anime, from the first show I watched (Dragon Ball Z) to my current favorite (Jujutsu Kaisen). It includes a table of the anime I watched, a list of other favorites, images, and links to explore more.

## What I Learned
I learned how to structure a webpage with HTML, including using headings, paragraphs, images, lists, tables, and links. I also practiced adding a navigation bar and using HTML attributes like `bgcolor` and `<font>` for colors.

## Challenges
Some challenges included:
- Making sure the text was visible on a black background.
- Learning the proper placement for the navigation bar.
- Understanding how to add color and background using HTML-only techniques.
